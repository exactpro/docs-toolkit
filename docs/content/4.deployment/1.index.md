# Deployment

Generate static HTML files from your documentation with `npx nuxt generate` command and it will be ready to be deployed to the web.

## Publishing to the web

There are several ways to deploy documentation to the web. 
Check out the documentation for details:

1. [Your own webserver (NGINX)](/deployment/web-server-nginx)
2. [Netlify](/deployment/netlify)
3. [GitHub pages](/deployment/github-pages)
4. [GitLab pages](/deployment/gitlab-pages)
