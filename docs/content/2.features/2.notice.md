# 1. Notice

The **Notice component** (`components/content/Notice.vue`) is created to highlight some part of the text.

It supports 4 color modes:
- default (gray)
- info (blue)
- note (yellow)
- warning (red)

## Usage

Default:

```md
::notice

  Your notice text

::
```

Result:

::notice

  Your notice text

::


Info:

```md
::notice{info}

  Your info text

::
```

Result:

::notice{info}

Your info text

::

Note:

```md
::notice{note}

  Your note text

::
```

Result:

::notice{note}

Your note text

::

Warning:

```md
::notice{warning}

  Your warning text

::
```

Result:

::notice{warning}

  Your warning text

::
