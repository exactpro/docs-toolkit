Docs template

Welcome to Exactpro docs template! It was made to help you in building documentation website for your project

Feel free to ask questions if any: docops@exactpro.com

Quick start

Requirements: NodeJS LTS 16+

Install dependencies:

npm install

Run local server:

npm run dev

Enjoy writing with Nuxt Content.

Static hosting

To generate HTML files for static hosting:

npm run generate

Generated files can be located in:

  • dist
  • .output/public

Content creation

When the website successfully deployed, you can start creating content. Website content generated from .md files, located in content directory of the project. In addition to markdown features like bold and italics, you can use MarkDown Components for highlighting important info.
